# Fixed-Rate Stacks Bonds

A decentralized finance (DeFi) protocol built on the Stacks blockchain using Clarity 4. This project enables users to issue and purchase fixed-term bonds that pay a fixed interest rate upon maturity.

The protocol is designed to provide predictable returns for investors and a predictable cost of capital for issuers, differentiating it from traditional variable-rate yield protocols like Stacking.
